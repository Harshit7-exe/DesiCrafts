# DesiCrafts Recommendation Models

This module implements the three AI/ML models specified in the project report:

1. Collaborative Filtering
2. Content-Based Filtering
3. Hybrid Recommendation

## Files

- `data/products.csv`: starter artisan product catalog
- `data/interactions.csv`: starter implicit user interaction dataset
- `recommender.py`: model classes
- `train_models.py`: training and artifact export
- `artifacts/`: trained model output after running the trainer

## Train

```bash
python ml/train_models.py
```
